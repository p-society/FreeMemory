Core Features

HMD v2 Decay Algorithm

Brain-inspired memory decay that strengthens with access and weakens over time, mimicking human forgetting patterns.

Memory Sectors

Hierarchical organization with contextual groups like Work, Personal, and Learning, each with custom decay rates.

Graph-Based Waypoints

Multi-hop knowledge traversal with semantic, temporal, and causal relationships between memories.

Vector Search

hnswlib-powered approximate nearest neighbor search for sub-millisecond semantic similarity queries.

RAG Context Retrieval

Advanced retrieval combining vector similarity, memory strength, and graph traversal for optimal context.

Type-Safe Architecture

Built with Bun.js, Drizzle ORM, and TypeScript for enterprise-grade reliability.
